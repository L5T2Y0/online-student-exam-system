方式一：最简单 - 使用自动脚本 ⭐推荐
========================================

1. 先在GitHub创建新仓库：
   - 访问 https://github.com/new
   - 仓库名：online-exam-system
   - 描述：一个功能完整、安全可靠的在线考试系统
   - 选择 Public 或 Private
   - ❗不要勾选任何选项（README、.gitignore、License）
   - 点击 "Create repository"
   - 复制仓库地址

2. 双击运行：快速上传.bat

3. 按提示输入你的GitHub仓库地址

4. 如果提示输入密码，使用Personal Access Token：
   - 访问 https://github.com/settings/tokens
   - 生成新token（勾选repo权限）
   - 复制token作为密码使用


方式二：使用GitHub Desktop（最直观）
========================================

1. 下载GitHub Desktop：https://desktop.github.com/

2. 打开GitHub Desktop，登录你的账户

3. File -> Add local repository

4. 选择项目文件夹

5. 点击 "Publish repository"
